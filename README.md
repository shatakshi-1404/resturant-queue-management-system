# Smart Queue Management System for Restaurants (MERN + TypeScript)

## Overview

This project allows customers to **join restaurant queues online** and enables restaurant owners to **manage queues digitally**. The system supports **real-time queue updates**, token generation, and a **restaurant dashboard** with live queue analytics.

Built with **MERN stack + TypeScript + Socket.io + Vite**.

---

## Features

### Customer
- Register/Login
- View restaurants
- Join queue online
- Receive token
- Live queue status
- Cancel token

### Restaurant Owner
- Login
- Dashboard to view queue
- Call next token
- Track queue analytics
- TV display for "Now Serving"

### Advanced
- JWT Authentication
- Real-time queue using Socket.io
- QR code / SMS notification (optional)
- Multi-branch support

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS  
- **Backend:** Node.js, Express, TypeScript, MongoDB, Socket.io  
- **Authentication:** JWT  
- **Other:** Axios, Context API, React Router v6
